 NOTE:venvとvirtualenv
 元はvirtualenvでその一部の機能がvenvとしてpythonの標準に取り込まれた。なので、virtualenvの方が少し多機能。

# install

```bash 
pip install virtualenv
```


# make virtual environment

```bash
virtualenv env_name
```
または
```bash
python -m venv env_name
```
# activate(on Windows)
```bash
env_name\Script\activate
```

### deactivate
```bash
deactivate
```
