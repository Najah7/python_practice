# テキストからHTMLへの変換

## 課題
プレンテキストに書式を付与したい。
（プレーンテキスト→HTML）
条件：
1. 入力は内容に関係ないコードやタグが埋め込まれている必要はないものとする。
1. 見出し、段落、リスト項目、インラインのテキストなど各種のブロックを処理できるようにする。
1. 今回の実装ではHTMLを扱うが、他のマーク合アップ言語に容易に拡張できるようにする。

## 役に立つツール
- ファイルの読み書き、または少なくとも標準入力(sys.stdin)からの読み込みとprintによって出力
- 入力を１行ごとに反復する
- 数種の文字列関連メソッド
- ジェネレータをいくつか作成
- reモジュール
